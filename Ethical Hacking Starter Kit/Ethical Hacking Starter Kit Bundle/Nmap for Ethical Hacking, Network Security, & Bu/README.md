# Nmap for Ethical Hacking, Network Security, & Bug Bounties

This course covers network scanning and ethical hacking using Nmap. Starting with the basics of TCP/IP and OSI models, you'll learn about TCP, UDP packets, and network topologies. Practical demonstrations include port scanning, Wireshark packet analysis, and Nmap scripting. Topics also include target selection techniques, service version detection, OS detection, Nmap output formats, and bypassing firewalls. The course concludes with creating a Python program for scanning and OS detection using Nmap. By the end of the course, you'll be skilled in using Nmap for ethical hacking.
