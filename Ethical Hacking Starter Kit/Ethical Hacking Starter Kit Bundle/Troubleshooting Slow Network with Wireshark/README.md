# Troubleshooting Slow Network with Wireshark

Wireshark is a powerful, free, and open-source network traffic analyzer. It allows users to examine network packets at a deep level, making it an invaluable tool for troubleshooting and cybersecurity. This course focuses on mastering Wireshark basics on Windows, covering live network capture, filtering, protocol analysis, and problem identification. By the end of the course, participants will have the skills to effectively troubleshoot errors in proprietary protocols and address TCP issues.
